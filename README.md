# Deploy a TICK architecture onto your Raspberry Pi.
Deployment package

## INSTALL
- Telegraf 
- InfluxDB 
- Chronograf 
- Kapacitor 

### Procedure

```bash
git clone https://github.com/TamataOcean/TICK-For-Raspberry-Pi.git

```
launch 
```bash
INSTALL.sh
```

### chronograf interface
Launch your navigator http://RaspiAddress:8888/
